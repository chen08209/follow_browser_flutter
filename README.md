# follow_browser_flutter

一个flutter浏览器的demo

受限于官方对webview的支持以及flutter平台的限制,后续会转向原生Android

## 特色

深色模式

material3 动态主题

简洁明快的ui设计

完善的多标签页管理

便捷的书签导入
